*** Springboot(Gradle) + React AWS 배포하기 ***

* Springboot(Gradle) 빌드
- cmd 열기
- 스프링부트 파일의 디렉토리로 이동
- 입력 #gradlew build
- 해당 폴더\build\libs 에서 .jar 파일 생성 확인
- cmd에서 #java -jar 방금 생성된 .jar 파일명.jar 입력하여 실행
- 종료 시 ctrl +c 

* React 빌드
- cmd 열기
- 스프링부트 파일의 디렉토리로 이동
- 입력 #npm run build
- 해당 폴더\build 폴더 생성 확인
